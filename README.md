# AI Full Stack Application

A full-stack application that integrates React and Spring Boot to provide AI-powered features like chat, image generation, and recipe suggestions using OpenAI.

---

## 🚀 Features

- Generate images from text prompts  
- Chat with an AI assistant  
- Generate recipes based on ingredients, cuisine, and dietary preferences  
- Clean tab-based UI for switching between features  

---

## 🛠 Tech Stack

- **Frontend:** React, JavaScript  
- **Backend:** Spring Boot, Spring AI  
- **AI:** OpenAI (Chat + Image models)  

---

## 📸 Demo

Includes an interactive UI for:
- Image generation  
- Text/chat generation  
- Recipe generation
